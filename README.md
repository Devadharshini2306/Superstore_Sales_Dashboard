# Superstore Sales Dashboard using Power BI

## Project Overview

This project presents an interactive sales dashboard created using Power BI to analyze Superstore sales data.
The dashboard helps visualize key business metrics such as sales, profit, and regional performance.

## Objectives

* Analyze sales and profit data
* Identify top-performing regions and categories
* Visualize trends over time
* Create an interactive dashboard using filters and slicers

## Tools & Technologies

* Power BI
* Microsoft Excel / CSV Dataset

## Dataset

The dataset used is the **Superstore Sales dataset**, which includes:

* Order Date
* Region
* Category & Sub-Category
* Sales
* Profit

## Dashboard Features

* **KPI Cards** for Total Sales and Total Profit
* **Bar Chart** for Sales by Region
* **Pie Chart** for Sales by Category
* **Bar Chart** for Profit by Sub-Category
* **Line Chart** for Sales Trend over Time
* **Slicers/Filters** for interactive analysis

## Steps Performed

1. Loaded dataset into Power BI
2. Cleaned and transformed data using Power Query
3. Created multiple visualizations
4. Added slicers for interactivity
5. Designed and formatted dashboard layout
6. Derived insights from visualizations

## Key Insights

* Certain regions generate higher sales than others
* Technology category contributes significantly to revenue
* Some sub-categories have low or negative profit
* Sales show an increasing trend over time

## Output

* Interactive Power BI Dashboard
* Visual insights from sales data
